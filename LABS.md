# 🧪 Laboratórios de Estudo

Foco em experimentos de performance, baixo nível e validação de simplicidade.

---

## ☕ Java Performance Lab
- **Object Layout:** Entender o overhead de objetos Java na memória.
- **GC Tuning:** Comparar o comportamento de diferentes coletores.
- **Microbenchmarking:** Uso do JMH para medir latência.

## 🐹 Go Efficiency Lab
- **Concurrency Patterns:** Worker pools e rate limiters.
- **Allocations Analysis:** Minimizar escapes para o heap.
- **Network I/O:** Servidores de alta performance com Std Lib.

## 🅰️ Angular Performance Lab
- **Change Detection Mastery:** OnPush e Signals.
- **Bundle Optimization:** Code-splitting e lazy loading.
- **Web Workers:** Processamento pesado em background.

## 🛠️ Essential Systems Lab
- **Minimalist Linux:** Containers Alpine e Scratch.
- **Network Fundamentals:** TCP/IP e TLS overhead.
- **Storage Performance:** Estratégias de indexação.
