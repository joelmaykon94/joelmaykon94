# 🗓️ Roadmap Especialista 2026

Este roadmap detalha o plano de 365 dias focado em **Java, Go e Angular**, com uma filosofia de arquitetura minimalista e foco total em performance e controle.

---

## 🏗️ Estrutura Trimestral

### 🗓️ Q1: Java Performance (Jan - Mar)
- **JVM Deep Dive:** Garbage Collection (ZGC, Shenandoah), JIT Compilation, profiling com JFR.
- **Project Loom:** Concorrência eficiente com Virtual Threads.
- **Spring Boot Enxuto:** Uso consciente do ecossistema Spring.

### 🗓️ Q2: Go Simplicity (Abr - Jun)
- **Go Fundamentals:** Concorrência (Goroutines, Channels), Interfaces.
- **Low Level Go:** Pacote `unsafe`, gerenciamento de memória.
- **Standard Library:** Construir serviços minimizando dependências externas.

### 🗓️ Q3: Angular Enterprise (Jul - Set)
- **Angular Moderno:** Signals para gerenciamento de estado reativo.
- **Performance Web:** Otimização de renderização, Core Web Vitals.
- **RxJS Essencial:** Uso inteligente de streams.

### 🗓️ Q4: Arquitetura Minimalista (Out - Dez)
- **KISS:** Design focado em simplicidade.
- **DevOps Essencial:** Docker, Kubernetes (o básico para rodar).
- **System Design:** Evoluir sistemas sem complexidade acidental.
