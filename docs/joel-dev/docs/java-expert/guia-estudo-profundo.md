# ☕ Guia de Estudo: Especialista Java Moderno

Tornar-se um especialista em Java hoje não é apenas conhecer a sintaxe, mas entender o ecossistema que evoluiu drasticamente desde o Java 8. Este guia é baseado no [Roadmap.sh/Java](https://roadmap.sh/java) e foca no **estudo profundo** e prático.

:::info
Objetivo
Este guia visa transformar um desenvolvedor "que sabe Java" em um **Especialista em Java Moderno**, capaz de projetar sistemas de alto desempenho, escaláveis e resilientes.
:::
